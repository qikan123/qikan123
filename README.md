### 介绍

因偶然之下了解到现在国内期刊网站乱象，随便搜索一个期刊，在搜索引擎第一个页面里的网站有中介的有造假诈骗的，很难第一时间找到官网，甚至有的期刊就没有官网。如果你是一个刚接触这个行业没人指点，一下子就不知所措了。

国内期刊搜索做的比较大的就是[知网](https://navi.cnki.net/knavi/)与[万维书刊网](https://www.eshukan.com/)，知网的问题是期刊搜索并不是它的一级页面，你需要在里面跳转多个页面之后才能找到自己想要的，这不利于搜索引擎抓取信息。

万维书刊网的问题是知名度不够，搜索权重不够高，不能与其他鱼龙混杂的网站区分开来，但是优点在于里面的信息更丰富且有一个评论区，可以看到部分作者对于期刊的评价。

另外一个搜索方向就是微信小程序以及公众号，很多期刊都运营了微信小程序和公众号，如果你找不到期刊官网那么可以去微信搜一搜。微信的问题就是对搜索引擎完全封闭，自成一派。

本项目旨在为那些对发论文有需求的萌新指引一个明确的方向，希望能通过Github本身在搜索引擎里的高权重以及后续大家持续的支持，使本项目成为一个“期刊导航”。

### 使用

**总表与附加表**：总表为全部论文期刊，附加表为文学故事、小说、报纸、集刊，直接使用总表加载时间可能会比较长，在页面中使用**Ctrl+F**进行页面搜索

**拼音划分**：提供一个期刊名称，按首字母拼音前往对应的页面，在页面中使用**Ctrl+F**进行页面搜索。如果提供的期刊名称是曾用名，则需要去原始Excel表里查询

**学科划分**：如果你还想要看看相同学科下其他期刊，则可以根据期刊名称判断学科，或者是通过期刊的CN号码后面的分类找到对应学科，CN号可通过拼音表查询获得，本项目学科划分依据就是按照CN号码后面的类别

	CN号格式：CN地区号-序号/分类号

### 期刊收录规则

收录的期刊满足三个条件：中文、有CN号、影响因子相对较高

### 表格说明(包括原始Excel表)

**拼音表**：按照名称拼音排序

**学科表**：按照复合影响因子降序排列（**注意**：这个项目从六月末开始做，到十二月末结束，影响因子更新了一波。最开始收录的期刊影响因子是24年的，后面的是25年的，但是一年之内影响因子的波动不太明显，期刊的相对位置变化较小，有时间我再把之前的影响因子更新一波，问题不大）

**ISSN**：除《佛学研究》外全部都有，期刊的国际号码

**CN**：全部都有，期刊的国内号码，有CN号才是国内的正经期刊

**期刊名称**：仅写了期刊的现用名称，如果想查询曾用名则需要去原始Excel表格在注解中查找

**学科**：这里用的不是CN号分类而是知网分类，知网的分类并不是特别精确，看看就行

**复合影响因子**：跨学科，更为全面

**综合影响因子**：学科内，更专业

**官网**：有的期刊没有官网，有的期刊有好几个，在有多个官网时会将更重要的放在前面。如果有多个官网的同时还有多个作者入口，那么官网与作者入口会按照顺序一一对应

**作者入口**：有这个说明可以直接在线投稿，需要注意的是有的期刊没有官网，只有一个作者入口

**邮箱**：有的邮箱可以投稿，有的只能咨询。如果没有官网以及作者入口，那么必定是邮箱投稿，在这种情况下如果有多个邮箱分类投稿，会将全部的邮箱都展示出来，否则最多展示两个以节省空间

**注解**：在Excel表格中用来标注额外信息，大部分是期刊的曾用刊名

表中期刊最少会有作者入口和邮箱中的一个，保证能够具有网络交互的方式（中央美术学院的《美术研究》是唯一的例外，此刊只收邮寄的纸质稿）

### 写在最后

我本身并不是科研相关从业人员，知道这个事情之后搜索了一下，看着谷歌搜索首页上乱七八糟的网址感觉莫名的气愤，当时我就在想应该有这么一个期刊导航一样的东西，不需要很复杂，只需要给一个期刊名返回一个官网地址而已。于是就有了这个项目，一次收录一点，花了半年时间断断续续的收录了三千多个期刊。每一个期刊的网站我都进去看了一圈，确定没啥问题再收到表里。

即使我花了很多时间在这上面，但是国内期刊总数上万，我不能保证把所有应该收录的期刊都写了下来，我也不能保证这三千多条数据一点问题也没有，所以如果你发现了这个项目的数据有问题，例如我遗漏了某个核心期刊，或者某个期刊的网址不对，或者随着时间推移期刊的数据发生了改变，请你在issue中提出来，和我一起完善这个项目。另外，由于我是从一个程序员而不是一个从业者的角度去看这个问题，可能对于期刊这个问题的理解或者处理方式或者表格的设计有不足之处，如果你对此有什么想法，可通过邮件联系我。
